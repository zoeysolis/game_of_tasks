
“El desgaste y la falta de seguimiento en las aplicaciones de productividad tradicionales”
En el ámbito estudiantil, la organización del tiempo, la gestión de tareas y el seguimiento de
los objetivos son muy importantes para alcanzar el éxito. Comúnmente, se recurre a
herramientas de productividad como listas de tareas (To-Do List) o calendarios. Pero, el uso
de las mismas provoca el mayor problema: la falta de motivación. Interactuar con estas
interfaces se siente como una extensión de la carga, más en nuestros tiempos donde hay
una manía por el scrolling y la recompensa rápida. Por consecuencia, el estudiante tiene
frustración y abandona la herramienta volviendo a los mismos ciclos de procrastinación y
desorganización formando un círculo vicioso.

2. Propuesta de solución
“Gamificación mediante un Dashboard de Misiones de Vida”
Para resolver esta falta de motivación, proponemos el desarrollo de GAme of Task. Se trata de una app interactiva que transforma la gestión del tiempo en mecánicas
de juego de rol tradicional (rpg)
En vez de visualizar una lista rígida de obligaciones, las tareas del estudiante se configuran
como “misiones”
